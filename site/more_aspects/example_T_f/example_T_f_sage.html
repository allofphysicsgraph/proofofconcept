The following carries out the actions but does not invoke the inference rules: 
T,f=var('T,f')

expr= T == 1/f

expr=expr*f

expr=expr/T

print(expr)
The implementation below checks that the claimed step as stated by the inference rule is carried out correctly: 
T=var('T')
f=var('f')
# latex input: T = 1/f
input_expr = (1)/(f) == (T)
# latex output: T f = 1
expected_output_expr = T * f = 1
# latex feed: f
feed = f
input_expr * feed == expected_output_expr

More Aspects

  
 

There are lots of aspects that have been explored that may be useful but that haven't been fully explored. This page is a collection of the avenues that could be investigated.

An example (shown graphically on the front page) is of the relations between frequency (f) and period (T). In functional form, 

" T = 1/f " = declareInitExpr()
" T f = 1 " = multiplyBothSidesBy ( " T = 1/f "," f " )
" f = 1/T " = divideBothSidesBy( " T f = 1 "," T " )
declareFinalExpr( " f = 1/T " )
which is a subset of frequency relations.

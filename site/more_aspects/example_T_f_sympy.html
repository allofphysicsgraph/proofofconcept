The following carries out the actions of the derivation but does not explicitly use the inference rules: 

T,f = symbols('T f')
expr_LHS = T
expr_RHS = 1 / f
expr_LHS = expr_LHS * f
expr_RHS = expr_RHS * f
expr_LHS = expr_LHS / T
expr_RHS = expr_RHS / T
Then the new expression is 
>>> expr_LHS
f
>>> expr_RHS
1/T

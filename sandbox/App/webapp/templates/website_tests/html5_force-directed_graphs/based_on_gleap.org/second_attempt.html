Description

This is a HTML5 canvas-based JavaScript implementation of Fruchterman and Reingol's force based graph visualization algorithm.

Controls

You can add new nodes by left clicking within the simulation.

Resources

Want to do something with force based graph visualization or HTML5 canvas? The source code running this simulation is open source (MIT license) and moderately documented. Also check out some of these links:
